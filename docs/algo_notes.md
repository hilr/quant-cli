# 算法调研笔记

## 波峰波谷识别

### 1. 滚动窗口极值法
- 前后各 N 天为最高 → 波峰；最低 → 波谷
- 简单但噪声多，参数敏感

### 2. Zigzag
- 最小回撤阈值（如 5%），反转超过阈值才确认转折
- 直观但滞后

### 3. Prominence 突出度（推荐）
- `scipy.signal.find_peaks(prominence=X)`
- 波峰 = 比两侧谷底高出一定幅度的点
- 参数含义明确，对噪声不敏感

### 4. 持久同调
- 记录极值被"填平"所需的最小回撤
- 一次扫描出全部极值及强度

## 震荡区间识别

### 1. 波峰波谷法
- 连续高点接近 + 连续低点接近 → 横盘
- 高点振幅 < 3%，低点同理

### 2. 布林带收窄
- `boll_bandwidth = (upper - lower) / mid`
- 带宽处于低位分位 → 震荡蓄势

### 3. 价格区间重叠率
- 滑动窗口内每日区间与整体区间的重合度
- 重合度 > 阈值 + 振幅 < 阈值 → 震荡

### 4. 赫斯特指数
- H < 0.5 → 均值回归（震荡）
- 需长窗口

### 5. ATR 振幅比较
- 短期振幅 / 长期 ATR < 阈值 → 波动率收缩
