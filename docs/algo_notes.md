# 算法调研笔记

## 波峰波谷识别

### 1. 滚动窗口极值法
- 前后各 N 天为最高 → 波峰；最低 → 波谷
- 简单但噪声多，参数敏感

### 2. Zigzag
- 最小回撤阈值（如 5%），反转超过阈值才确认转折
- 直观但滞后

### 3. Prominence 突出度（推荐）
- `scipy.signal.find_peaks(prominence=X)`
- 波峰 = 比两侧谷底高出一定幅度的点
- 参数含义明确，对噪声不敏感

### 4. 持久同调
- 记录极值被"填平"所需的最小回撤
- 一次扫描出全部极值及强度

## 震荡区间识别

### 1. 波峰波谷法
- 连续高点接近 + 连续低点接近 → 横盘
- 高点振幅 < 3%，低点同理

### 2. 布林带收窄
- `boll_bandwidth = (upper - lower) / mid`
- 带宽处于低位分位 → 震荡蓄势

### 3. 价格区间重叠率
- 滑动窗口内每日区间与整体区间的重合度
- 重合度 > 阈值 + 振幅 < 阈值 → 震荡

### 4. 赫斯特指数
- H < 0.5 → 均值回归（震荡）
- 需长窗口

### 5. ATR 振幅比较
- 短期振幅 / 长期 ATR < 阈值 → 波动率收缩

## 布林带通道策略优化探索（CSI300，2026-06）

### 背景
CSI300（000300）触下轨买入策略 60 日胜率仅 ~50%，最大浮亏 −30%，与 512890 红利低波（93% 胜率，最大 −5%）天差地别。原因：CSI300 高 Beta，下轨信号在熊市里是飞刀。

### 已尝试方向与结论

**1. 减去 MA 做去趋势（detrended BB）**
- `residual = close - MA(120)`，对 residual 做 BB(120) ± 1.5σ
- **结论**：数学上等价于原方案。`σ(close) ≈ σ(close - MA)` 在同窗口下几乎相等（CSI300 上 126 vs 125）。去趋势只改变 Y 轴刻度，不改变信号。
- **效果**：胜率、信号数、最大浮亏与原方案一致。

**2. 动态 k = 1.5 + α × max(0, -slope_120) × 100（手调版）**
- `slope_120 = (MA[t] - MA[t-120]) / MA[t-120]`，MA 下行时按斜率放大 k
- α=2, slope_w=120 是最佳组合
- **效果**：信号数 999 → 331（过滤 67%），胜率 51% → 55%，60d 均值 +0.6% → +1.7%，最大浮亏 −37% → −31%
- **重要观察**：α=2 × 100 这个系数过强，k_eff 实际近似阶跃函数 —— slope 一旦 < −3%，k_eff 跳到 7.5+，下轨基本封死。38% 的日子 k_eff > 10。**几乎等价于「MA 上行才买」二元规则**（即 `tag_rising_ma`）。
- **盲点**：2007 大顶前 MA 还在涨时，动态 k = 1.5，没起过滤作用。

**3. ML 选择信号（logistic / RF / GB + walk-forward CV）**
- 特征：slope_20/60/120、σ_now、dev_z、rsi_14、sig_density_60、ret_vol_20
- 标签：60d 后 close/entry - 1 > 0（二分类）
- 走样 CV：train 过去 5-10 年，test 下一年
- **结论**：所有模型都跑不赢手调 α=2：
  - Logistic C=0.5：胜率 48%，60d 均值 +1.2%
  - RF depth=5：胜率 53%，+1.2%
  - GB depth=2：胜率 53%，+1.5%
  - 手调 α=2：胜率 61%，+5.0%（基准）
- **评价口径缺陷（重要）**：当时用「60d > 0」二分类 + 胜率做主指标，太粗。应该改成：
  1. 回归目标（直接预测 60d 收益率）
  2. 接入 `run_signal_strategy` 做真策略回测（NAV / Sharpe / 最大回撤）
  3. 跟「手调 α=2」基准对比，而不是「全买」基准

### 未尝试方向（待评估）

**A. 动态窗口 W**
- 波动率 regime 切换：σ 高用短窗（60），σ 低用长窗（250）
- 周期分析（Hurst / FFT）找主导周期
- 直觉：熊市反应快用短窗，牛市慢牛用长窗

**B. 动态 k（带宽自适应）**
- 波动率分位数：`k_eff = σ_history.quantile(0.95) / σ_current`，保持 5% 触发率
- **绝对偏离阈值**：`buy = close ≤ MA × (1 - 5%)`，去 σ 化，跨 regime 含义一致
- 历史命中率校准（过拟合风险大）

**C. 自适应布林带（学术版）**
- `MA_eff = α × EMA + (1-α) × MA_prev`，α 随波动率调整
- 参数空间大，过拟合风险高

### 关键反思

CSI300 的根本问题不是「窗口不够长」或「带宽不够宽」，而是**触下轨本身就是熊市特征**。动态参数能让信号更挑剔，但不能让本质失效的信号变有效。512890 之所以好用，是因为红利低波 + 长期上行通道，触下轨 = V 型反弹；CSI300 高 Beta 触下轨 = 趋势下跌中段。

### 推荐尝试顺序（如果回来继续做）

1. 先试「绝对偏离阈值 5%」（最简单，跟当前动态 k 对比，看 σ 自适应重不重要）
2. 如果有效，叠加趋势过滤（α=2）
3. 如果还有空间，再试 regime 切换 W
4. 不推荐 C（参数太多）
