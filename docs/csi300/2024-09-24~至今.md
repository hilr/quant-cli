# CSI300 牛市段波动率特征（2024-09-25 ~ 至今）

> 区间起点取 2024-09-24（政策利好引爆日，次日 +8.5% 跳空）。
> 数据源：`/mnt/dataset/index_quote_history/000300.parquet`，共 415 个交易日。
> 起点 close 3401.53 → 终点 4891.71，区间涨幅 **+43.8%**。

---

## 1. 日收益率分布

| 指标 | 牛市段 | 此前全历史 |
|---|---|---|
| 样本数 | 416 | 5512 |
| 日均收益 | +0.099% | +0.029% |
| 日标准差 | 1.25% | 1.57% |
| **年化波动率** | **19.9%** | 25.0% |
| 最小单日 | −7.05% | — |
| 最大单日 | +8.48% | — |
| **偏度 skew** | **+0.28** | −0.24 |
| **峰度（超峰）kurt** | **+10.6** | +4.3 |
| 年化夏普 | 1.38 | 0.29 |
| Jarque-Bera | 2122, p=0（强拒绝正态） | — |

### 关键特征

- **正偏 +0.28**：与历史负偏（−0.24）方向相反。牛市里向上跳空比向下跳空多。
  - 涨天 225 个（54.2%），跌天 189 个（45.5%），平 1 个。
  - 平均涨天 +0.84%，平均跌天 −0.78%，涨/跌绝对均值比 = 1.07。
- **超峰 +10.6**：厚尾极重（正态 = 0）。比历史 4.3 还厚很多 —— 牛市伴随跳空行情。
  - 极端值都集中在启动期：2024-09-24 后启动日 +8.5%、2024-10-08 见顶回调 −7% 等。
- **波动率本身下降**：年化 19.9% < 历史 25.0%。单边上涨压制了日间噪声。
- **Jarque-Bera 强烈拒绝正态**：BB 通道假设的近似正态在此期间严重失真，1.5σ 触发率会显著偏离理论 6.7%。

## 2. 滚动年化波动率分布

| 窗口 | 样本 | 均值 | 中位 | 10% 分位 | 90% 分位 | 偏度 | 峰度 |
|---|---|---|---|---|---|---|---|
| 10 日 | 406 | 15.73% | 14.74% | 7.34% | 22.77% | +3.20 | +16.24 |
| 20 日 | 396 | 15.75% | 14.70% | 9.24% | 22.81% | +2.35 | +10.29 |
| 60 日 | 356 | 15.59% | 15.82% | 9.96% | 18.86% | +0.99 | +6.04 |

### 关键特征

- **强右偏**：短窗尤其严重（10 日偏度 +3.2）。跳空瞬间把 10 日 vol 顶到 **74%**（max），均值被极端值拉高。
- **窗口越短，分布越尖厚**：10 日峰度 16.2，60 日峰度 6.0。短窗对突发跳空极敏感。
- **中位数稳定**：三个窗口的中位 vol 都在 14.7%~15.8%，但右尾差距大 —— 90 分位从 18.9% 飙到 22.8%。

## 3. 对通道策略的启示

- **固定 k 的 BB 通道在波动率飙升时会被频繁假突破**：vol 突变期间 σ 急速膨胀，下轨被拉得很低，价格还没到下轨趋势就已经在跌了。
  - 这与 algo_notes 中「CSI300 触下轨在熊市是飞刀」是同一现象的另一面。
- **正偏意味着「等量对称通道」低估上行**：上轨按 +kσ 设、下轨按 −kσ 设，在正偏分布里上行突破概率 > 下行突破概率。
- **厚尾意味着极端单日频繁**：±7% 的单日行情在正态假设下应是 ~5σ 罕见事件，实际却多次出现。

## 4. 复现命令

```bash
uv run python << 'EOF'
import polars as pl
import numpy as np
from scipy import stats

df = (
    pl.read_parquet('/mnt/dataset/index_quote_history/000300.parquet')
    .with_columns(pl.col('date').str.to_date('%Y-%m-%d'))
    .sort('date')
    .filter(pl.col('date') >= pl.lit('2024-09-24').str.to_date('%Y-%m-%d'))
    .with_columns((pl.col('close') / pl.col('close').shift(1) - 1).alias('ret'))
    .drop_nulls('ret')
)
ret = df['ret'].to_numpy()
print(f"mean={ret.mean()*100:.4f}%  std={ret.std(ddof=1)*100:.4f}%  ann_vol={ret.std(ddof=1)*np.sqrt(252)*100:.2f}%")
print(f"skew={stats.skew(ret):+.4f}  kurt={stats.kurtosis(ret):+.4f}")
EOF
```
