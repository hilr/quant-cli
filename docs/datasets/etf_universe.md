# etf_universe — ETF 范围清单（宽基/行业/商品/债券/境外市场五类）

| 项 | 内容 |
|---|------|
| 命令 | `uv run python -m quant.cli etf-universe <input_dir> <output_dir>` |
| 输入 | `{input_dir}/{code}.parquet`（fund_quote_adjusted） |
| 输出 | `{output_dir}/etf_universe.parquet`（单文件汇总） |
| 逻辑 | 遍历基金行情，筛出名称含「ETF」的基金，按名称关键词分为宽基指数/行业/商品/债券/境外市场五类，并给宽基与行业 ETF 标注子分类 |
| 字段 | code, name, category, subcategory |

**字段含义：**
| 字段 | 含义 |
|------|------|
| code | 基金代码 |
| name | 基金名称 |
| category | 大类：宽基指数ETF / 行业ETF / 商品ETF / 债券ETF / 境外市场ETF |
| subcategory | 子分类：仅宽基/行业两类有值，商品/债券/境外为 null。宽基按跟踪指数（沪深300/中证A500/上证50…），行业按细分行业（证券/芯片/光伏…） |

**大类分类规则（按优先级，先匹配的生效，均不匹配归「行业ETF」）：**

| 大类 | 命中关键词 |
|------|-----------|
| 债券ETF | 债、国债、国开 |
| 商品ETF | 黄金、原油、白银、豆粕、有色ETF、油气 |
| 境外市场ETF | 恒生、恒指、港股、港美、香港、H股、中概、美股、美国、海外、日经、纳斯达克、纳指、标普、德国、法国、英国、印度、东南亚、日本、越南、韩国、中韩、新加坡、欧洲、金砖、东盟、亚太、全球、QDII、沪港深、沪港通 |
| 宽基指数ETF | 沪深300、HS300、上证50、中证500、中证1000、中证2000、国证2000、中证800、A500、A50、MSCI、科创50、科创100、科创板、创业板、深证、等权、300、50ETF、500ETF |
| 行业ETF | 以上均不匹配的 ETF（消费、医药、军工、新能源、芯片、银行、证券、煤炭等） |

> 「300」作为宽基兜底关键词，覆盖沪深300的策略变体（300增强、300价值、300成长、300红利、300ESG、300等权、300现金流等），它们本质仍跟踪沪深300宽基指数。

**宽基回判规则：** 若某 ETF 命中宽基关键词、但名称同时含行业/主题词（消费、医药、军工、新能源、芯片、半导体、人工智能、光伏、汽车、银行、证券、煤炭、钢铁、化工、有色等），则降级为**行业ETF**。典型如「创业板新能源ETF」「科创板人工智能ETF」。

**子分类规则：** 仅对宽基/行业两类标注。宽基按跟踪的宽基指数（如沪深300、中证A500、上证50、科创50），行业按细分行业（如证券、芯片、光伏、锂电池）。同名基金管理人的不同产品（如「证券ETF景顺」「证券ETF广发」）会落到同一子类——同一子类的基金应基本同涨同跌、相关度较高。商品/债券/境外市场 ETF 的 subcategory 为 null（不细分）。
