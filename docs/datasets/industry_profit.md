# industry_profit — 工业企业每月利润总额

| 项 | 内容 |
|---|------|
| 命令 | `uv run python -m quant.cli industry-profit --data-path <readonly> --output-dir <dataset>` |
| 输入 | `{data_path}/gov_stats/工业企业指标/{year}.csv`（≤2024）或 `.xlsx`（≥2025） |
| 输出 | `{output_dir}/gov_stat/industry_profit/{year}.csv`，每年一个文件（2000–2026 共 27 个） |
| 逻辑 | 把「利润总额累计值(亿元)」按月差分为「每月当月利润」，相邻已知累计点之间均摊差额 |
| 字段 | year, month, profit |

源数据是统计局公布的**累计值**（且每年 1 月恒为空）。本数据集将其转换为**每月当月值**。

**字段含义：**
| 字段 | 含义 |
|------|------|
| year | 年份 |
| month | 月份 1–12 |
| profit | 当月利润总额（亿元），不可得时为空 |

## 计算口径

把已知累计点视作阶梯，相邻两点之间的月份**均摊**差额：

```
锚点 cum[0] = 0
months_known = sorted(cum 的月份)
prev_m, prev_cum = 0, 0.0
for m in months_known:
    gap = m - prev_m                      # (prev_m, m] 区间内的月份数
    每个月 prev_m+1..m 的 profit = (cum[m] - prev_cum) / gap
    prev_m, prev_cum = m, cum[m]
# 最后一个已知月之后的月份（如稀疏年的 12 月）→ 空
```

- **gap=1（满数据年 2000–2006、2010–2025）**：每月 = `cum[m] − cum[m-1]`，**精确值**
- **gap=2**（每年年初 1–2 月）：1、2 月各 = `cum[2]/2`（统计局只公布 1–2 月合并累计）
- **gap=3（仅 2007–2009）**：3/4/5 月各 = `(cum5−cum2)/3`，6/7/8 月各 = `(cum8−cum5)/3`，9/10/11 月各 = `(cum11−cum8)/3`，**等额估计值**
- **无上界的月份**（2007–2009 的 12 月、2026 的 5–12 月）→ 空

## 各年覆盖

| 年份 | 累计来源 | 已知月 | 产出覆盖 |
|------|---------|--------|---------|
| 2000–2006 | 自身 `利润总额累计值` | 2–12 | 1–12 月全（精确） |
| 2007–2009 | 自身（旧口径，仅 2/5/8/11） | 2,5,8,11 | 1–11 月（1/2精确、3–11 等额估计）、12 月空 |
| **2010** | **2011 文件 `利润总额上年同期累计值`**（新口径重建） | 2–12 | 1–12 月全（精确，新口径） |
| 2011–2025 | 自身 `利润总额累计值` | 2–12 | 1–12 月全（精确） |
| 2026 | 自身（仅 2/3/4） | 2,3,4 | 1–4 月、5–12 月空 |

### 2010 重建说明

2010 自身文件只有 2/5/8/11 月且为旧口径；本数据集改用 **2011 文件的「利润总额上年同期累计值」**（=2010 同月累计，但为 2011 新口径重述值，比 2010 旧口径偏低约 2–3%）。这样 2010 与 2011+ 同属新口径、连续可比；2010 自身旧口径值不用。

### 2007–2009 等额估计提示

2007–2009 年源数据仅在 2/5/8/11 月公布，3–11 月单月值无法精确还原，本数据集用「季度桶等额拆分」（如 3/4/5 月各 = `(cum5−cum2)/3`）作为**估计值**，使用时需知悉其非真实月度统计。

源数据详细坑（月份列乱序、2011 口径调整、2025/2026 是 XLSX 等）见 [gov_stats_industrial.md](gov_stats_industrial.md)。

## 使用示例

```bash
uv run python -m quant.cli industry-profit --data-path /mnt/readonly_dataset --output-dir /mnt/dataset
```
