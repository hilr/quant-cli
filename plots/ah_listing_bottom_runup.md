# ah_listing_bottom_runup — A+H 上市前 N 交易日 A 股底部 → 反弹最高点 形态分析

`ah_listing_bottom_runup.py` 的文档与结果。回答：「每家公司港股上市日 D
之前 N 个交易日内（默认 120 ≈ 半年），A 股的最低点出现在距 D 多少个交易日？
低点到之后的最高点反弹了多少？反弹花了多久？上市日相对最高点的位置？」

## 用法

```bash
# 默认 120 交易日
uv run python plots/ah_listing_bottom_runup.py \
  --input-xlsx /path/to/ah_companies.xlsx

# 改窗口（如 250 交易日 ≈ 1 年）
uv run python plots/ah_listing_bottom_runup.py \
  --input-xlsx /path/to/ah_companies.xlsx \
  --window 250
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--input-xlsx` | A+H 公司清单 xlsx（必填） | — |
| `--adjusted-dir` | A 股前复权行情目录 | `/mnt/dataset/stock_quote_adjusted` |
| `--window` | 上市前回看窗口长度（交易日，默认 120 ≈ 半年） | `120` |
| `--as-of` | 只保留 港股上市日 <= 此日期 的公司（YYYY-MM-DD） | 今天 |
| `--output-png` | 输出 PNG | `/mnt/dataset/ah_listing_bottom_runup.png` |
| `--output-csv` | 输出明细 CSV | `/mnt/dataset/ah_listing_bottom_runup.csv` |

## 口径

- 卖出日 = D（港股上市日），用 D 当天或之前最近一个 A 股交易日的收盘价。
- 窗口 = `[list_idx − window + 1, list_idx]`，共 ≤ window 个交易日。
- **min_idx** = 窗口内 `argmin(close)`，最低点。
- **max_idx** = `[min_idx, list_idx]` 区间内 `argmax(close)`，即最低点之后到上市日之间的最高点（含上市日）。
- 派生指标：
  - `min_to_list_days` = `list_idx − min_idx`（最低点距上市日的交易日数）
  - `min_to_max_days` = `max_idx − min_idx`（低→高持续交易日数）
  - `max_to_list_days` = `list_idx − max_idx`（最高点距上市日的交易日数；0 = 上市日即最高）
  - `runup_pct` = `close[max_idx] / close[min_idx] − 1`（%）
  - `list_vs_peak_pct` = `close[list_idx] / close[max_idx] − 1`（%，负值 = 上市日已回调）

## 输出

- **PNG** 2×3 面板：
  - 左上 / 中上 / 右上：`min_to_list_days`、`min_to_max_days`、`runup_pct` 直方图
  - 左下 / 中下：`max_to_list_days`、`list_vs_peak_pct` 直方图
  - 右下：`min_to_list_days` vs `runup_pct` 散点（带线性拟合 + ρ）
- **CSV**：每行一家公司，含 min/max/list 日期、价格、各距离与收益率
- **控制台**：5 项形态指标的分布统计 + 最低点时机累计分布 + "上市日即峰值"占比

## 参考结果（34 家公司，window=120，截至 2026-07-19）

清单来自清洗后的 `/mnt/dataset/ah_companies_clean.xlsx`。原始用户表 49% 行
A 股代码错配（详见 `ah_listing_runup.md` 的"数据清洗说明"），清洗后剩 34 家
确认已上市、A 股代码正确的公司。

### 5 项形态指标分布

| 指标 | N | 均值 | ±95%CI | 中位 | σ | P5 | P25 | P75 | P95 |
|---|---|---|---|---|---|---|---|---|---|
| 最低点距上市日（交易日） | 34 | 67.2 | (54.6, 79.8) | **71** | 37.5 | 12 | 35 | 101 | 118 |
| 低点 → 最高点（交易日） | 34 | 52.6 | (40.5, 64.6) | **39** | 35.8 | 10 | 22 | 82 | 114 |
| 最高点距上市日（交易日） | 34 | 14.6 | (7.6, 21.7) | **5** | 21.0 | 0 | 1 | 20 | 52 |
| 低点 → 最高点 收益率（%） | 34 | **+69.8** | **(+41.6, +98.1)** | +36.0 | 84.0 | +11 | +24 | +80 | +282 |
| 上市日 vs 最高点（%） | 34 | **−9.9** | (−13.0, −6.8) | **−6.1** | 9.2 | −29 | −15 | −4 | 0 |

### 最低点时机累计分布

| 距上市日 | 公司数 | 占比 |
|---|---|---|
| ≤ 10 交易日 | 1 | 2.9% |
| ≤ 20 交易日 | 6 | 17.6% |
| ≤ 30 交易日 | 7 | 20.6% |
| ≤ 60 交易日 | 16 | **47.1%** |
| ≤ 90 交易日 | 23 | **67.6%** |
| ≤ 120 交易日 | 34 | 100% |

### 三个核心发现

1. **底部集中出现在上市前 3-4.5 个月**：中位 71 个交易日（≈ 3.4 个月），
   P25-P75 为 [35, 101]。68% 的公司最低点落在上市前 90 个交易日内。
   只有 2.9%（1 家）的最低点紧贴上市日（≤ 10 个交易日）。

2. **反弹幅度两极分化严重**：均值 +70%、中位 +36%、P95 高达 +282%。
   从底部到最高点的中位持续期约 2 个月（39 个交易日）。极端样本：
   鼎泰高科（+352%）、芯碁微装（+314%）、三环集团（+265%）等 PCB /
   半导体光刻主题股仍是反弹冠军。均值被这些大反弹股拉高，中位更能代表
   "典型"样本。

3. **峰值离上市日非常近**：中位只有 5 个交易日！34 家里 4 家（11.8%）
   上市日就是窗口最高点；P75 也只有 20 个交易日。**典型形态：底部
   出现在上市前 ~3.5 个月，反弹持续 ~2 个月，在上市前 1 周见到 A 股高点**。
   上市日相对峰值中位回调 6.1%，P25 回调 15%。

### 散点：底部时机 vs 反弹幅度

`min_to_list_days` 与 `runup_pct` 几乎不相关（ρ 接近 0）。底部出现得早 vs 晚
对反弹幅度没有显著线性关系——反弹幅度主要由行业 / 主题 / 个股弹性决定。

## 解读

- **可操作的抄底窗口**：上市前 60-90 个交易日（≈ 上市前 3-4.5 个月）是
  典型的底部区间，68% 的公司最低点都出现在这里。
- **最佳止盈点**：上市前 1 周（中位 max_to_list_days = 5）。如果持有到
  上市日，平均要回吐 6.1% 的涨幅（中位），最坏情况回吐 29%。
- **反弹 ≠ 大牛市**：window=120 时反弹中位只有 +36%（vs window=250 时的
  +80%），说明更短窗口看到的是"上市前 3-6 个月内的修复行情"，而不是
  完整的多年趋势。如果想吃完整趋势，参考 `--window 250` 的版本。

## 与 250 交易日窗口的对比

| 指标 | 120 交易日 | 250 交易日 |
|---|---|---|
| 最低点距上市日（中位） | 71 | 229 |
| 低→高 收益率（中位） | +36% | +80% |
| 最高点距上市日（中位） | 5 | 46 |
| 上市日 vs 峰值（中位） | −6.1% | −13.4% |

短窗口看到的是"上市前的最后一波修复"——底部更近、反弹更小、峰值离上市日
更近；长窗口看到的是"过去 1 年的完整趋势"——底部远、反弹大、但峰值也
更早出现。两个视角互补。

## 局限

- **未剔除大盘 beta**：上市前的反弹部分来自大盘整体上涨，未做指数调整。
- **max 永远 >= min**：因为 max 取的是 min 之后的最高，不会出现"先高后低
  单边下跌"的情况（这种情况下 min 接近窗口末尾、max 也在末尾附近、
  runup 接近 0）。
- **样本结构**：34 家公司集中在 2024-2026 港股牛市 + AI 算力 / PCB / 半导体主题，
  反弹分布有明显的主题性偏差。
