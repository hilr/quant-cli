# annual_volatility — 沪深300 每年年化波动率 + 指数对照

`annual_volatility.py` 的完整文档与结果快照。结果随数据更新，由脚本控制台输出誊写。

## 用法

```bash
# 沪深300，默认 √252 年化、20 日滚动
uv run python plots/annual_volatility.py --output /mnt/dataset/annual_volatility_000300.png

# 只打印表格不画图
uv run python plots/annual_volatility.py

# 改年化因子或滚动窗口
uv run python plots/annual_volatility.py --trading-days 244 --roll-window 60 --output /mnt/dataset/annual_volatility_000300.png
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--index-dir` | 指数 parquet 父目录 | /mnt/dataset/index_quote_history |
| `--code` | 指数代码 | 000300（沪深300） |
| `--trading-days` | 年化因子（每年交易日数） | 252 |
| `--roll-window` | 滚动波动率窗口（交易日） | 20 |
| `--output` | 输出 PNG 路径（不指定则只打印表格） | None |

## 输出

控制台年度表格 + 双面板 PNG（上=沪深300 收盘价；下=年化波动率：蓝色阶梯=每年年化波动率，橙色细线=滚动20日年化波动率，红色虚线=全期均值）。

## 方法

- **年化波动率**（close-to-close realized vol）= 每日对数收益率 `ln(close_t / close_{t-1})` 的样本标准差 × √`trading_days`
- 按日历年分组，每年一个值；标准差用 ddof=1（样本标准差）
- 滚动波动率 = 滚动窗口内日收益率标准差 × √`trading_days`，显示年内波动率演变
- 年化因子 252 为国际通用惯例；A股实际每年约 242~244 个交易日，可按需调整

## 统计

- 全部 25 年：均值 **23.02%**，中位 **21.16%**
- 完整年份（≥200 交易日，24 年）：均值 **23.27%**，中位 **21.19%**
- 最高 **2008 年 48.51%**（金融危机），最低 **2017 年 10.15%**（慢牛低波）

## 观察

- **极端高波**：2008（48.5%，金融危机）、2015（39.8%，杠杆牛/股灾）、2007（37.1%，大牛市）、2009（32.8%，危机后反弹）
- **低波年份**：2017（10.2%）、2023（13.5%）、2025（15.4%）——多为窄幅震荡/慢牛
- 波动率与价格：高波年往往对应剧烈涨跌（2008 暴跌、2015 先暴涨后股灾），低波年对应平稳走势
- 滚动波动率能更细地捕捉到年内波动率突变（如 2020-03、2024-09 的尖峰）
