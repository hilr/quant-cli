# annual_volatility — 沪深300 每年年化波动率 + 指数对照

`annual_volatility.py` 的完整文档与结果快照。结果随数据更新，由脚本控制台输出誊写。

## 用法

```bash
# 沪深300，默认 √252 年化、20 日滚动
uv run python plots/annual_volatility.py --output /mnt/dataset/annual_volatility_000300.png

# 只打印表格不画图
uv run python plots/annual_volatility.py

# 改年化因子或滚动窗口
uv run python plots/annual_volatility.py --trading-days 244 --roll-window 60 --output /mnt/dataset/annual_volatility_000300.png
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--index-dir` | 指数 parquet 父目录 | /mnt/dataset/index_quote_history |
| `--code` | 指数代码 | 000300（沪深300） |
| `--trading-days` | 年化因子（每年交易日数） | 252 |
| `--roll-window` | 滚动波动率窗口（交易日） | 20 |
| `--output` | 输出 PNG 路径（不指定则只打印表格） | None |

## 输出

控制台年度表格（两口径）+ 双面板 PNG（上=沪深300 收盘价；下=年化波动率：蓝阶梯=close-to-close，绿阶梯=ATR，橙细线=滚动20日，红虚=close-to-close 全期均值）。

## 方法

两种口径，均按日历年分组、× √`trading_days` 年化：

- **close-to-close**：每日对数收益率 `ln(close_t / close_{t-1})` 的样本标准差（ddof=1）× √N
- **ATR（真实波幅）**：每日 True Range / close 的年内均值 × √N
  - True Range = `max(high−low, |high−prev_close|, |low−prev_close|)`
  - 含日内振幅 + 隔夜跳空，捕捉的价格波动比 close-to-close 更全

> 注意：ATR 是均值绝对波幅（非标准差），× √N 年化是与 close-to-close 对齐的近似口径，便于横向比较；两口径的相对大小反映「日内来回 vs 净趋势」的结构差异。

滚动波动率 = 滚动窗口内日收益率标准差 × √N，显示年内波动率演变。年化因子 252 为国际通用惯例；A股实际每年约 242~244 个交易日，可按需调整。

## 观察

- **极端高波**（两口径一致）：2008（c2c 48.5% / ATR 64.4%，金融危机）、2015（39.8% / 49.8%，杠杆牛/股灾）、2007（37.1% / 49.5%，大牛市）、2009（32.8% / 42.7%，危机后反弹）
- **低波年份**：2017（c2c 10.2%）、2023（13.5%）、2025（15.4%）——窄幅震荡/慢牛
- **ATR vs close-to-close**：多数年份 ATR > close-to-close（日内来回波动大于净涨跌）；但 2002-2004、2025-2026 出现 close-to-close ≥ ATR，反映这些年份以**单边趋势**为主（日内单方向移动，收盘净变动接近甚至超过日内振幅）
- 滚动波动率能更细地捕捉到年内波动率突变（如 2020-03、2024-09 的尖峰）
