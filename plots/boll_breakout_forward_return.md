# boll_breakout_forward_return — Bollinger 通道突破事件 → 未来收益分布

对应脚本：`plots/boll_breakout_forward_return.py`
输出：`/mnt/dataset/boll_breakout_fwd_{code}_w{window}_k{k}.png` + 控制台统计

## 图说明

把 N 日 Bollinger 通道（MA ± k·σ）的"首次穿越"事件挑出来，统计未来 5/10/20 日
收益的分布，回答：

> 「沪深300 收盘价突破 Bollinger 上/下轨后，未来 5/10/20 日的收益分布
> 长什么样？是否有显著的均值偏移或动量/反转效应？」

输出 2 行 × 3 列直方图网格：
- **上行**：向上突破（close > 上轨）—— 5d / 10d / 20d 直方图
- **下行**：向下突破（close < 下轨）—— 5d / 10d / 20d 直方图

每面板：
- 红柱正收益、绿柱负收益
- 橙虚线 = 事件均值、蓝点线 = 中位数、灰点划线 = 全局基线均值
- 右上文本框：N / μ / α vs 基线 / M / σ / P(>0) / P5·P95

## 算法

- **通道**：`ma = close.rolling_mean(window)`、`sigma = close.rolling_std(window)`
  （Polars 默认 ddof=1）、`upper = ma + k·sigma`、`lower = ma − k·sigma`。
- **突破事件（首次穿越）**：前一日仍在通道内 + 当日 close > upper（向上）或
  close < lower（向下）。同方向连续多日在通道外只算 1 个事件。
- **未来收益**：`r_fwd_m = close[t+m] / close[t] − 1`。
- **基线**：所有交易日的 r_fwd_m 分布（无条件基线）。
- **alpha**：事件均值 − 基线均值。

## 怎么读

| 看点 | 含义 |
|------|------|
| 橙虚线在灰点划线右侧（向上突破） | 事件均值 > 基线，**正向 alpha**——动量延续 |
| 橙虚线在灰点划线左侧（向下突破） | 事件均值 < 基线，**负 alpha**——短期延续跌势 |
| α 随 horizon 单调放大 | 信号有持续性（如向上突破 α 从 5d → 20d 单调上升） |
| α 从负转正（向下突破） | 短期延续跌势 + 长期均值回归 |
| P(>0) 显著偏离 50% | 分布偏移的直观证据（>60% 或 <40% 算显著） |

## 用法

```bash
# 默认：沪深300 / 20 日 Bollinger ±2σ / 未来 5/10/20 日
uv run python plots/boll_breakout_forward_return.py

# 换通道参数
uv run python plots/boll_breakout_forward_return.py --window 30 --k 1.5

# 换未来窗口
uv run python plots/boll_breakout_forward_return.py --horizons 3 5 10

# 换标的
uv run python plots/boll_breakout_forward_return.py --code 000905 \
  --adjusted-dir /mnt/dataset/index_quote_history
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--code` | 标的代码 | 000300 |
| `--adjusted-dir` | 含 `{code}.parquet` 的行情目录 | /mnt/dataset/index_quote_history |
| `--window` | Bollinger 通道窗口（日） | 20 |
| `--k` | σ 倍数 | 2.0 |
| `--horizons` | 未来持有期（日），可传多个 | 5 10 20 |
| `--bucket-width` | 直方图桶宽（小数，0.01 = 1%） | 0.01 |
| `--output` | 输出 PNG 路径 | boll_breakout_fwd_{code}_w{window}_k{k}.png |

## 数据源

- `{adjusted-dir}/{code}.parquet`：标的日收盘（`date, close` 两列即可）。

## 参考结果（沪深300 / 20 日 ±2σ / 截至 2026-07-17）

| 方向 | N | 周期 | 事件 μ | 基线 μ | α | P(>0) 事件 | P(>0) 基线 |
|---|---|---|---|---|---|---|---|
| 向上突破 | 149 | 5d | +0.84% | +0.18% | **+0.66%** | 60.4% | 52.5% |
| | | 10d | +1.47% | +0.36% | **+1.12%** | 56.4% | 52.5% |
| | | 20d | +1.93% | +0.73% | **+1.20%** | 52.0% | 51.8% |
| 向下突破 | 156 | 5d | −0.14% | +0.18% | **−0.32%** | 50.0% | 52.5% |
| | | 10d | −0.37% | +0.36% | **−0.73%** | 42.2% | 52.5% |
| | | 20d | +0.14% | +0.73% | −0.59% | 46.1% | 51.8% |

**核心读数**：

1. **向上突破是有效的短期动能信号**：α 单调放大（+0.66% → +1.12% → +1.20%），
   P(>0) 在 5d 达 60.4%——历史胜率显著高于基线。
2. **向下突破 5~10 日内不要抄底**：α 持续为负（−0.32% → −0.73%），
   P(>0) 在 10d 只 42.2%；真正的反弹要到 20 日之后。
3. **不对称性**：A 股向上突破比向下突破信号更强、更稳定；向下突破是 V 型
   反转形态（短期跌、长期均值回归）。

## 口径提醒

- **首次穿越定义**：连续多日在通道外只算 1 个事件，避免重复计数。
- **滚动重叠窗口**：相邻事件高度自相关，做经验概率参考可以，严格统计推断
  需对有效样本数打折。
- **未做牛熊分段**：牛市/熊市下的突破信号特征可能不同，本图是全周期聚合。
