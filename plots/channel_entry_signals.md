# channel_entry_signals — 布林带下轨买入点 + 前瞻收益 / 浮亏标注

`channel_entry_signals.py` 的完整文档与结果快照。结果随数据更新，由脚本控制台输出誊写。

## 图说明

复用 `channel_backtest.run_backtest` 的下轨入场逻辑（`close ≤ MA(window) − k·σ`），画出价格通道（MA 中轨 + 上下轨 + 通道带）+ 收盘价，标记每一个买入点：

- **实心绿三角**：前瞻窗口已满（默认 60 日），标注为「窗口内最大涨幅 / 60 日收益」
- **空心绿三角**：前瞻窗口未满（最近 N 个买入点），用截至最新的已知数据补全，days 列显示实际使用的交易日数

每个三角上方标注两行：`+X.X%`（窗口内最大涨幅）和 `+Y.Y%`（窗口末端收益），绿涨红跌。

## 用法

```bash
# 默认：512890 红利低波，MA120 ± 1.5σ，60 日前瞻
uv run python plots/channel_entry_signals.py

# 自定义参数
uv run python plots/channel_entry_signals.py --code 512890 --window 120 --k 1.5 --fwd 60
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--code` | 基金/指数/股票代码 | 512890 |
| `--adjusted-dir` | 前复权行情目录 | /mnt/dataset/fund_quote_adjusted |
| `--window` | MA + σ 窗口（交易日） | 120 |
| `--k` | 带宽倍数（σ 倍数） | 1.5 |
| `--fwd` | 前瞻窗口（交易日，≈3 月） | 60 |
| `--output` | 输出 PNG 路径（默认 channel_entries_{code}.png） | None |

**输出：** 双栏 — 上图 = 通道带 + 收盘价 + 买入点标注；控制台 = 每个买入点的明细表 + 聚合统计。

## 算法

- **买入信号** = `close ≤ MA(window) − k·σ(window)`（`tag_boll_lower`，见 `quant/tags.py`）
- **入场价** = 信号触发当日收盘价（实际持仓从下一日开始，T-1 信号 → T 持仓）
- **通道**：center = `MA(window)`，upper = `MA + k·σ`，lower = `MA − k·σ`
- **前瞻收益** = `close[t + fwd] / 入场价 − 1`
- **最大涨幅** = `max(close[t..t+fwd] / 入场价 − 1)`，见顶日 = 取最大值的相对偏移
- **最大浮亏** = `min(close[t..t+fwd] / 入场价 − 1)`（从入场价起算，不是 peak-to-trough，反映买入后真实最大账面损失）
- **窗口未满处理**：`j_end = min(t + fwd, 最新交易日)`，用 `[t, j_end]` 已有数据计算；`actual_days = j_end − t`，`is_partial = actual_days < fwd`
- **聚合统计**（胜率 / 均值 / 中位数）**只用完整窗口的买入点**，避免被样本不足的最近点带偏
- 数据源 = 前复权收盘价（`fund_quote_adjusted`），消除分红除权影响

## 怎么读

| 信号 | 含义 |
|---|---|
| 实心绿三角 | 窗口已满，标注为最终值 |
| 空心绿三角 | 窗口未满，按已知数据估算，仍会随新数据更新 |
| 标注上排 `+X.X%` | 窗口内最大涨幅（衡量上限潜力） |
| 标注下排 `+Y.Y%` | 窗口末端收益（衡量实际持有结果） |
| 控制台 `days` 列 | 实际使用的交易日数，< fwd 即窗口未满 |

## 示例结果（512890 红利低波，截至 2026-07-08）

共 18 个买入点，14 个完整窗口，4 个窗口未满用最新已知值补。

### 完整窗口 14 笔（聚合统计依据）

- 60 日胜率：**13/14（93%）**，收益均值 **+8.9%**，中位数 **+10.4%**
- 最大涨幅均值 **+12.3%**，中位数 **+13.2%**（最差 +4.7%）
- 最大浮亏均值 **−1.8%**，最差 **−5.1%**，见底均值 **9.6 日**
- **5 笔**买入后从未浮亏，**9 笔**先浮亏后转盈，唯一下跌的（2022-08-03）仅 −0.6%
- 最大涨幅均值（12.3%）大于 60 日收益均值（8.9%），说明多数买入后经历先涨后回调，而非单边持有
