# conditional_forward_return — 前 N1 日收益分桶 → 之后 N2 日收益 条件分布

`conditional_forward_return.py` 的文档与结果。**和 `next_day_return_heatmap` 互补**：
后者看整体聚合的 ρ / P(↑|↑)（会被中间大量横盘样本主导、抹平尾部信号），
本工具固定"前 N1 日收益"在某个范围内（例如大跌之后），看"之后 N2 日收益"的
**条件分布**是否有明显的均值偏移或厚尾。

## 用法

```bash
# 默认：1 → 1 + 沪深300 + large_segments.csv + 默认桶
uv run python plots/conditional_forward_return.py

# 3 日 → 3 日（条件分布最有信号的窗口）
uv run python plots/conditional_forward_return.py --today-window 3 --next-window 3

# 自定义桶边界（近 0 更细、尾部更宽）
uv run python plots/conditional_forward_return.py \
  --today-window 3 --next-window 3 \
  --bucket-edges "-inf,-10,-6,-3,-1,0,1,3,6,10,inf"

# 换中尺度分段
uv run python plots/conditional_forward_return.py \
  --segments-csv /mnt/dataset/csi300_regime_segments/medium_segments.csv
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--segments-csv` | 分段 CSV | `/mnt/dataset/csi300_regime_segments/large_segments.csv` |
| `--quote-path` | 指数日行情 parquet | `/mnt/dataset/index_quote_history/000300.parquet` |
| `--code` | 指数代码（标题用） | `000300` |
| `--today-window` | 截止当日的 N1 日收益窗口长度 | `1` |
| `--next-window` | 之后的 N2 日收益窗口长度 | `1` |
| `--bucket-edges` | 桶边界（%，逗号分隔，支持 -inf/inf） | `-inf,-8,-5,-3,-1,0,1,3,5,8,inf` |
| `--output` | 输出 PNG 路径 | `conditional_forward_return_{code}_{N1}b_{N2}f.png` |

**输出：** PNG（左右两面板）+ 控制台牛熊两张条件统计表（含全局基线）。

## 默认桶边界

```
(-∞, -8%)  [-8%, -5%)  [-5%, -3%)  [-3%, -1%)  [-1%, 0%)
[0%, +1%)  [+1%, +3%)  [+3%, +5%)  [+5%, +8%)  [+8%, +∞)
```

**设计思路**：近 0 更细（绝大多数样本集中在此）、尾部更宽（极端样本稀少但重要）。
两端用 `±inf` 把所有越界值都收进来。桶边界固定不随 N1 变化，便于跨窗口对比。

## 图说明

- **x 轴**：前 N1 日收益桶（10 个）。
- **y 轴**：之后 N2 日收益（%）。
- **箱线图**（须=P5/P95、箱=P25/P75、黑横线=中位）。
- **红点 + 红误差棒**：桶均值 ± 95% 置信区间（`1.96 × std/√n`）。
- **橙虚线**：全局基线均值。
- **n<5 的桶**：跳过画图（样本太少统计不靠谱），但仍在控制台输出。
- **桶顶 n=...**：每桶样本数（提示统计可靠性）。

## 判定阈值（粗略经验）

- **均值偏移显著**：桶均值 95%CI 不包含全局基线均值。
- **厚尾**：峰度（excess kurtosis，正态分布 = 0）>> 0。
- **偏态**：偏度 |skew| > 0.5 视为明显偏向。

## 算法

1. 构造配对 `(r_back_t, r_fwd_t)`，同 `next_day_return_heatmap`。
2. 按 `--bucket-edges` 用 `np.digitize` 分桶（半开 `[lo, hi)`）。
3. 每桶统计：
   - N / 均值 / 标准差 / 偏度 / 峰度（excess，numpy 公式 `m4/(m2²)−3`）
   - P5 / P25 / P50 / P75 / P95
   - 均值标准误 `std/√n` → 95%CI = `mean ± 1.96 × se`
4. 按 regime（牛/熊）分别统计，含全局基线对照。

## 怎么读

| 看点 | 含义 |
|------|------|
| 红点显著偏离橙虚线 | 该桶条件下，未来收益均值有系统偏移（择时机会） |
| 须线（P5/P95）异常长 | 该桶条件下，未来收益波动剧烈、厚尾（择时风险） |
| 红误差棒不含橙虚线 | 均值偏移统计显著（不是噪声） |
| 桶之间均值呈 U 形 | 两端条件 → 同向延续；中间横盘 → 继续横盘 |
| 桶之间均值呈反向 | 一端反弹、另一端回落（均值回复） |
| 牛 vs 熊同桶对比 | 同一前期条件下，regime 带来的基线差异 |
| 峰度 > 3 | 该桶条件下，未来收益有厚尾（极端事件比正态多） |

## 解读

- **聚合 ρ 会掩盖尾部信号**：3→3 整体 ρ=+0.037 看起来"动量信号弱"，
  但条件分布视角下，**牛市两端桶（[-8%,-5%)、[+5%,+8%)）的均值偏移都显著
  且偏移 0.7-0.9pp**，被中间 1000+ 个横盘样本稀释掉了。
- **熊市暴跌反弹是非常强的信号**：前 3 日跌 >8% 后，未来 3 日均值 +2.02%
  （n=33、95%CI 不含 0），偏移 +2.33pp 是所有桶里最大的。这是单看 ρ=+0.011
  完全发现不了的择时机会。
- **U 形而非单调**：牛市不是简单的"前涨→后涨"动量，而是**两端 → 同向延续**、
  **横盘 → 继续横盘甚至小跌**。中期动量策略应避开 [0%, +1%) 这种低波动区。
- **厚尾集中在两端**：牛市右尾 [+5%, +∞) 峰度异常高，提示"前涨后大涨"的
  少数极端样本主导了均值偏移——择时时要意识到分布是非对称厚尾，
  不能只看均值。
- **实盘含义**：
  - 熊市里"前 3 日跌 >8%"是高置信度的反弹买入信号（CI [+0.17, +3.87]）。
  - 牛市里"前 3 日处于两端"是延续信号，"前 3 日几乎不动"反而是低收益区。
  - 单看整体 ρ 会低估这些机会；本工具的核心价值就是把聚合统计压平的信号
    在条件维度重新展开。

## 局限

- **桶样本不均**：两端桶（[-∞,-8%)、[+8%,+∞)）n 通常 <50，95%CI 较宽，
  统计功效有限。可用 `--bucket-edges` 调整或拉长 N1 增加尾部样本。
- **峰度对极端值敏感**：n<30 的桶峰度估计噪声大，要谨慎解读。
- **同 `next_day_return_heatmap` 的局限**：重叠窗口自相关、跨段边界窗口不剔除、
  最后一段未完结。
