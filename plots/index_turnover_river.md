# 沪深300 / 中证500 / 中证1000 / 其他 · 成交额占比 river 图

对应脚本：`plots/index_turnover_river.py`
输出：`/mnt/dataset/index_turnover_river.png`

## 图说明

streamgraph（sym 基线，整体恒为 100%），把两市总成交额按市值档位拆成 4 条带；
右轴叠加沪深300 收盘（灰淡线），对照占比结构与大盘走势的相关性。

| 档位 | 指数 | 代码 | 覆盖 |
|------|------|------|------|
| 大盘 | 沪深300 | 000300 | 市值前 300 |
| 中大盘 | 中证500 | 000905 | 301–800 |
| 中小盘 | 中证1000 | 000852 | 801–1800 |
| 其他 | — | — | 更小微盘、ST、及上述未覆盖的股票 |

四档互不重叠，前 3 个中证指数合计覆盖约前 1800 名。「其他」即两市总额减去三者后的余量。

## 计算口径

```
total    = sh_turnover(000001 上证综指) + sz_turnover(399106 深证综指)
other    = total − (hs300 + csi500 + csi1000)
各档占比 = 各档 turnover / total × 100%
```

**分母为何用 399106 而非 399001**：深证成指 399001 仅 500 只成分股，其 turnover 在
2010 年及以前远小于深市全市场（例：2010-01-04 399001=142 亿 vs 399106=823 亿），
拿它当分母会严重低估深市、使占比之和 > 100%。深证综指 399106 是深市综合指数，
turnover 始终对应该市场全部股票成交额之和，作分母才正确。上证综指 000001 同理覆盖沪市全市场。

中证系列指数虽 2007/2014 才挂牌，但 parquet 里的 turnover 自基准日 2004-12-31 起按
成分股回算，故全区间（2005-01-04 起）均可分解。按日 inner join 五个 turnover>0 的交易日。

## 怎么读

- **沪深300 带变宽**：资金向大盘蓝筹集中（抱团 / 风格切大盘 / 杠杆牛）；
- **中证1000 / 其他带变宽**：资金向中小盘、题材、微盘股扩散（普涨 / 中小盘行情）；
- **「其他」趋势性扩张**（近年升至 ~29%）：A 股扩容，三指数之外的小微盘成交占比上升。

## 区间特征

- 2005：市场高度集中，沪深300 单档 ~64%；
- 2010：随中小板/创业板扩容，沪深300 降至 ~43%，其他档抬升；
- 2015（杠杆牛）：沪深300 反弹至 ~63%（蓝筹领涨），其他被压缩；
- 2020 后：其他档趋势性走高至 ~29%，大盘集中度持续下行。

## 数据源

均来自 `/mnt/dataset/index_quote_history/`，由 `uv run python -m quant.cli index-quote` 生成：

- `000001.parquet`（上证综指，沪市全市场 turnover）
- `399106.parquet`（深证综指，深市全市场 turnover）
- `000300.parquet` / `000905.parquet` / `000852.parquet`（沪深300 / 中证500 / 中证1000）

中证2000 (932000) 不在 finance_sina 数据源中，故未纳入；如需补该档需先扩充数据源。
