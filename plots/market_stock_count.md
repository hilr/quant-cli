# 沪深两市每月新上市股票数 / 新增流通市值 vs 沪深300（三轴，月频）

对应脚本：`plots/market_stock_count.py`
输出：`/mnt/dataset/market_stock_count.png`

## 图说明

按每只股票在明细数据中的最早交易日（「首日」≈ IPO 日）归组到月：

| 轴 | 数据 | 形式 | 含义 |
|---|---|---|---|
| 左轴 | 当月新上市股票数 | 绿色柱 | **发行节奏**：IPO 扩容速度 |
| 右轴 | 当月新上市股票首日流通市值之和（亿元） | 蓝线 + 面积 | **募资/新增流通盘规模** |
| 次右轴（外偏） | 沪深300 收盘（日线） | 灰淡线 | **大盘表现对照** |

时间范围默认按沪深300的范围（2002-01-04 ~ 今）。

## 数据源

- 个股明细：`/mnt/dataset/stock_quote_history/*.parquet`（每股票一个文件，由
  `uv run python -m quant.cli stock-quote` 生成）。每只股票取其 parquet 中
  最早一行的 `date` 作为首日、该行的 `free_float_market_cap` 作为首日流通市值。
- 沪深300 收盘：`/mnt/dataset/index_quote_history/000300.parquet`（日线）。

### 口径与边界

- **排除北交所/三板**：code 首位为 4/8，或 920 段。原因：这部分明细数据覆盖不全
  （920 段自 2025-10 才入库），首日并非真实挂牌日，会形成假峰值；且北交所独立于
  「沪深两市」。
- **时间范围按沪深300**：默认 2002-01-04 起（沪深300 数据起点）。这同时避开了
  1992 年内的边界老股（明细数据自 1992-01-02 起，1992 内的「首日」其实是更早上市的
  老股被截断到 1992）。传 `--start-date` 可在沪深300范围内进一步收窄。
- 首日流通市值取**首日当天**的值（非当前最新）。

## 怎么读

- **柱（新股数）**：陡增段对应 IPO 加速期——2010 创业板开闸、2019-2020 科创板 +
  注册制、2022 注册制全面推开。
- **线（流通市值）**：高于柱的相对位置 = 当月有大蓝筹募资。典型如 2007-11
  （中国石油等）、2020-07（中芯国际）。
- **新股数与流通市值背离**：股数多但市值低 = 中小盘 IPO 为主；股数少但市值高 =
  巨无霸上市。
- **与沪深300 对照**：IPO 高峰常出现在牛市中后段（2007、2015、2020-2021）；
  熊市阶段 IPO 明显放缓（2008、2024）。

## 运行

```bash
uv run python plots/market_stock_count.py
# 可选：--data-dir <path>（默认 /mnt/dataset/stock_quote_history）
#       --hs300-file <path>（默认 /mnt/dataset/index_quote_history/000300.parquet）
#       --output <path>
#       --start-date <YYYY-MM-DD>（默认按沪深300起始；传更晚日期可收窄）
#       --end-date <YYYY-MM-DD>（默认按沪深300最新；传更早日可收窄）
```
