# next_day_return_heatmap — 截止当日 N1 日 → 之后 N2 日 收益 2D 热力图（牛 vs 熊对比）

`next_day_return_heatmap.py` 的文档与结果。研究沪深300 **收益的自相关结构**：
过去 N1 日的累计涨跌幅，与之后 N2 日的累计涨跌幅，联合分布长什么样？
前涨后是延续还是反转？牛市和熊市里的这种关系有何差异？

## 用法

```bash
# 默认：N1=N2=1（即原"今天 → 明天"行为）+ 沪深300 + large_segments.csv
uv run python plots/next_day_return_heatmap.py

# 截止当日的 3 日收益 → 之后的 3 日收益
uv run python plots/next_day_return_heatmap.py --today-window 3 --next-window 3

# 只换一侧：截止当日 5 日 → 之后 1 日
uv run python plots/next_day_return_heatmap.py --today-window 5 --next-window 1

# 换中尺度分段
uv run python plots/next_day_return_heatmap.py \
  --segments-csv /mnt/dataset/csi300_regime_segments/medium_segments.csv

# 手动指定步长和范围（默认按 sqrt(max(N1,N2)) 自动缩放）
uv run python plots/next_day_return_heatmap.py --today-window 3 --step 0.5 --range 10

# 丢弃极端值而不是归入边缘桶
uv run python plots/next_day_return_heatmap.py --clip
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--segments-csv` | 分段 CSV | `/mnt/dataset/csi300_regime_segments/large_segments.csv` |
| `--quote-path` | 指数日行情 parquet | `/mnt/dataset/index_quote_history/000300.parquet` |
| `--code` | 指数代码（标题用） | `000300` |
| `--today-window` | 截止当日的 N1 日收益窗口长度（默认 1，即当天收益） | `1` |
| `--next-window` | 之后的 N2 日收益窗口长度（默认 1，即次日收益） | `1` |
| `--step` | 2D 直方图步长（%，0.5 = 0.5%）。默认按 sqrt(max(N1,N2)) 自动缩放 | 自动 |
| `--range` | 轴范围 ±N%。默认按 sqrt(max(N1,N2)) 自动缩放，超出范围归入边缘桶 | 自动 |
| `--clip` | 丢弃超出 `--range` 的样本（默认归入边缘桶） | off |
| `--output` | 输出 PNG 路径 | `segment_return_heatmap_{code}_{N1}b_{N2}f.png` |

**自动缩放规则**：未显式指定 `--step`/`--range` 时，按波动率 ∝ √N 估算：

| max(N1, N2) | step | range |
|---|---|---|
| 1 | 0.5% | ±7% |
| 2 | 0.5% | ±10% |
| 3-4 | 1.0% | ±12-14% |
| 5-7 | 1.5% | ±16-19% |
| 9-12 | 1.5% | ±21-24% |

**输出：** PNG（左右两面板，共享色阶）+ 控制台牛熊自相关/象限占比明细。

## 图说明

- **坐标**：x = 截止当日 N1 日收益（%）、y = 之后 N2 日收益（%），都按 `--step` 离散化。
- **颜色**：该格子占总样本的比例（%）。**log 色阶**——(0,0) 附近密度远高于尾部，
  线性刻度会把尾部糊成同色，log 才看得出尾部的细微差异。
- **白十字**（x=0 / y=0）：分四个象限。
- **象限标签**（面板内四角白底黑字）：
  - 右上 = **前涨后涨**（延续）
  - 左下 = **前跌后跌**（延续）
  - 左上 = **前跌后涨**（反转↑）
  - 右下 = **前涨后跌**（反转↓）
- **左上文本框**：ρ(back Nd, fwd Nd)、延续合计%、反转合计%、条件概率 P(后涨|前涨/跌)。
- **共享色阶**：牛/熊两面板用同一个 vmin/vmax，可直接对比色块强弱。

## 算法

1. 读 000300 前复权日 close，构造配对 `(r_back_t, r_fwd_t)`：
   - `r_back_t = close[t] / close[t-N1] - 1`（截止当日 N1 日收益）
   - `r_fwd_t  = close[t+N2] / close[t] - 1`（之后的 N2 日收益）
   - 首部丢 N1 行（无法算 r_back）、尾部丢 N2 行（无法算 r_fwd）。
2. 给每个 t 打 regime 标签（按段归属，半开 `[start, end)`，最后一段闭区间）。
3. 按 `bull` / `bear` 分组，做 2D 直方图（`np.histogram2d`），bins = `edges`。
4. 超出 `±range` 的样本默认归入边缘桶（`--clip` 改为丢弃）。
5. 统计：
   - ρ = Pearson(r_back, r_fwd)
   - 象限占比：把整个 (0, 0) 十字两侧的格子按象限求和
   - 条件概率：P(r_fwd > 0 | r_back > 0) 与 P(r_fwd > 0 | r_back < 0)

## 怎么读

| 看点 | 含义 |
|------|------|
| ρ ≈ 0 | 收益近似随机游走，择时几乎没优势 |
| ρ > 0（弱正） | 趋势延续：前涨后倾向继续涨（**动量效应**） |
| ρ < 0（弱负） | 均值回复：前涨后倾向回落（**反转效应**） |
| 主对角线（左下→右上）色深 | 延续主导（同涨同跌多） |
| 副对角线（左上→右下）色深 | 反转主导（涨跌交替多） |
| P(后涨\|前涨) vs P(后涨\|前跌) | 后者 > 前者 = 微弱反转效应；前者 > 后者 = 动量效应 |
| 牛 vs 熊的 P(后涨\|·) 整体差异 | 反映 regime 的基线涨跌概率（牛市里任意窗口都更易涨） |
| 窗口拉长后 ρ 变化 | N=1→3 等拉长窗口常暴露出单日噪声掩盖的趋势/反转信号 |

## 解读

### 1 日窗口：近似随机游走

- **日级别近似随机游走**：牛市 ρ=+0.02、熊市 ρ=+0.003，都贴近 0；
  延续/反转象限合计几乎对半（牛 50.5/49.5，熊 48.5/51.5）。
  **靠"昨天涨→今天继续涨"做单日择时，没有显著优势**。
- **微弱反转效应**：两个 regime 里 `P(后涨|前跌) > P(后涨|前涨)`——
  前日跌之后后日涨的概率，反而比前日涨之后后日涨的概率更高。
  熊市里这个差异更明显（49.0% vs 45.8%）。
- **regime 的基线概率才是主因**：牛市里不管前日涨跌，后日涨的概率都 ~57%；
  熊市里只有 ~47%。**段属性 >> 日间动量**——这也是为什么大级别分段
  比单日方向更有预测价值。

### 非对称窗口（N1→1）：前趋势 → 后单日，熊市短反转、N1=4 切换为动量

最贴近实盘择时的非对称窗口：用过去 N 日的累计涨跌作信号，预测下一个交易日。
N1=1 到 5 的反转强度轨迹（熊市 P(后涨\|前跌) − P(后涨\|前涨)）：

| N1→1 | 牛市 ρ | 熊市 ρ | 熊市反转强度 |
|---|---|---|---|
| 1 | +0.020 | +0.003 | **+3.2pp** |
| 2 | +0.005 | -0.018 | **+4.7pp** ← 峰值 |
| 3 | +0.014 | -0.006 | +2.5pp |
| 4 | **+0.034** | **+0.013** | +1.5pp（ρ 已翻正） |
| 5 | +0.024 | +0.003 | +2.9pp |

- **熊市反转信号在 N1=2 达峰**：4.7pp → N1=3 的 2.5pp → N1=4 的 1.5pp，
  短超跌反弹效应逐步耗尽。**熊市连跌 2 天后，次日反弹概率 49.6%**，
  是可操作的均值回复信号；3 天以上信号衰减。
- **N1=4 是反转 → 动量的切换点**：熊市 ρ 从 N1=3 的 -0.006 翻正到 +0.013，
  说明**跌满 4 天起进入趋势性下跌**——"连跌"信号从反转切换为延续，
  再追空比抄底更合理。
- **牛市 N1=4 反而出现动量信号**：ρ=+0.034，P(↑\|↑) 57.4% > P(↑\|↓) 55.9%
  （差 1.5pp）——**牛市连续 4 天涨之后，次日继续涨的概率略高**。
  但 N1=1/2/3/5 都没有这个效应，N1=4 像是孤峰，可信度待更多窗口验证。
- **牛市对前趋势整体不敏感**：N1=1/2/3/5 的 P(后涨\|·) 都贴近基线 ~57%，
  **牛市基线涨概率主导**，前几日涨跌几乎不影响明天。**牛市里别想靠趋势信号
  择时**，满仓比啥都强。
- **N1=5 几乎回到随机**：两个 regime 的 ρ 和概率差都缩小，信号被均值回归拉平。
- **实盘含义**：熊市超跌反弹窗口 **2 日最佳**，4 日失效，5 日以上反而要警惕
  趋势延续；牛市里 1-5 日的前期趋势几乎没有可用的次日择时信号（最多 N1=4
  有微弱动量）；**"短反转 + 中动量"的切换点大约在 3-4 日之间**。

### 3 日对称窗口：动量信号浮出水面

- **窗口拉长 → 自相关变强**：牛市 ρ 从 +0.020 升到 +0.037（接近翻倍），
  P(后涨|前涨) 从 56.6% 升到 **59.5%**。过去 3 日涨的样本，
  之后 3 日继续涨的概率显著高于基线（牛市基线 ~57%）。
- **熊市也翻转为延续主导**：延续合计从 48.5% → **51.1%**，
  P(后涨|前涨) vs P(后涨|前跌) 也从反转关系（45.8 < 49.0）翻转为
  动量关系（45.2 > 44.2）。
- **结论**：单日方向噪声太大，3 日累计才看得见"趋势延续"。
  做周度级别的动量/趋势策略比单日追涨杀跌更靠谱。
- **象限形态差异**：牛市右上一团很浓（同涨延续多），熊市左下一团很浓
  （同跌延续多）——视觉上能直接读出 regime 的偏移方向。

### 2 日窗口是个反常点

- **牛市 ρ 几乎归零**（+0.001），P(后涨|前涨) 58.0% 略高于 1d 的 56.6%，
  但 P(后涨|前跌) 58.4% 反而更高——**微弱反转**。
- **熊市 ρ=-0.023**，是 5 组里反转效应最强的一个：
  P(后涨|前跌) 44.9% > P(后涨|前涨) 44.4%。
- 1d、2d、3d 三个对称窗口结论不一致：1d 反转、2d 反转更明显、3d 翻转为动量。
  说明**短期窗口噪声大、信号方向不稳定**，3 日以上的中期窗口才看得清趋势延续。

## 局限

- **重叠窗口（重要）**：N1=N2=1 时相邻样本共享一个收益点，
  自相关程度可控；但 N>1 后相邻 anchor day t 的窗口 `(t-N1, t+N2)` 与
  `(t-N1+1, t+N2+1)` 大量重叠，**相邻样本高度自相关**——
  严格统计推断（置信区间、假设检验）需要谨慎，这里只做"经验概率"展示。
  N 越大，有效样本数越接近 `总天数 / max(N1, N2)`，而非名义 N。
- **跨段边界窗口不剔除**：anchor day t 决定归属，但 r_back 可能跨入上一段、
  r_fwd 可能跨入下一段。默认保留（更贴近实盘），如需"纯净段内"可自行加过滤。
- **大级别分段回顾式**：最后一段未完结（tail=true），实时不可知是否反转。
- **±range 之外归入边缘桶**：会人为堆高边缘格密度。如果关注极端尾部，
  可用 `--range` 扩展或 `--clip` 丢弃。
