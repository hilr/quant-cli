# segment_daily_return_distribution — 大级别牛熊分段内 日收益率分布

`segment_daily_return_distribution.py` 的文档与结果。把
`csi300_regime_segments/large_segments.csv` 的每一段大级别牛/熊区间，
逐日算沪深300 的日收益率 `(close - prev_close) / prev_close`，
画出 4×3 直方图网格 + 牛/熊聚合对比。

## 用法

```bash
# 默认：沪深300 + large_segments.csv + 0.5% 桶宽
uv run python plots/segment_daily_return_distribution.py

# 换中尺度 / 小尺度分段
uv run python plots/segment_daily_return_distribution.py \
  --segments-csv /mnt/dataset/csi300_regime_segments/medium_segments.csv \
  --output /mnt/dataset/segment_daily_return_dist_000300_medium.png

# 换桶宽（1%）
uv run python plots/segment_daily_return_distribution.py --bucket-width 0.01
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--segments-csv` | 分段 CSV | `/mnt/dataset/csi300_regime_segments/large_segments.csv` |
| `--quote-path` | 指数日行情 parquet | `/mnt/dataset/index_quote_history/000300.parquet` |
| `--code` | 指数代码（标题用） | `000300` |
| `--bucket-width` | 直方图桶宽（小数，0.005 = 0.5%） | `0.005` |
| `--output` | 输出 PNG 路径 | `/mnt/dataset/segment_daily_return_dist_{code}.png` |

**输出：** PNG（4×3 直方图网格）+ 控制台段明细表 + 牛 vs 熊聚合对比。

## 图说明

- **柱子**：每桶占比（%）。红 = 正收益、绿 = 负收益、灰 = 恰好 0（中国市场红涨绿跌）。
- **面板边框**：牛市红、熊市绿，做视觉分组。
- **橙虚线** = 均值；**蓝点线** = 中位数；**黑实线** = 0%。
- **面板标题**：`#段号 牛/熊(未完) 起止日期  N天  整段涨跌%  日均值`
  （`(未完)` 标记最后一段 tail=true 的未完结区间）。
- **右上文本框**：N、均值、中位、标准差、涨/跌占比、P1/P5/P95/P99 分位。

## 算法

1. 读 `large_segments.csv` 取分段（`seg, dir, size, trade_days, start_date, end_date, return_pct, tail`）。
2. 读 000300 前复权日 close，算 `(close - prev_close) / prev_close` 得到日收益序列。
3. 每段在区间内取日收益：**半开 `[start, end)`** 避免枢轴日重复计数（相邻段共享端点）；
   **最后一段尾段用闭区间 `[start, end]`**（后面没有下一段抢这个端点）。
4. 直方图桶边对齐到桶宽整数倍（0 是桶边）：`[-0.5%, 0%)` 归负、`[0, +0.5%)` 归正。

## 怎么读

| 看点 | 含义 |
|------|------|
| 牛市面板均值偏右、熊市偏左 | 大级别趋势方向直接体现在日收益一阶矩上 |
| 熊市标准差 > 牛市 | 熊市更躁动，极端单日涨跌都更多（fat tail 双侧） |
| 涨/跌占比对半（≈50/50） | 即使是牛市，日胜率也不会高到 70%+，趋势靠的是涨幅不对称 |
| P1 / P99 的绝对值 | 极端单日回撤 / 拉升的经验幅度，熊市左尾通常更深 |
| 整段涨跌% vs 日均值 | 整段涨跌 ≈ 日均值 × 天数 的复利版本，可相互验证分段合理性 |

**牛 vs 熊 聚合（段内日收益扁平合并）**：

- **牛市**：N=2806，均值 +0.194%，中位 +0.145%，标准差 1.423%，涨 56.8% / 跌 43.1%，P1/P99 -4.19% / +4.06%
- **熊市**：N=3138，均值 -0.110%，中位 -0.071%，标准差 1.646%，涨 47.5% / 跌 52.5%，P1/P99 -5.17% / +4.29%

## 解读

- **方向**：牛市日均 +0.19% vs 熊市日均 -0.11%，方向相反但绝对值不对称——
  牛市的"上推力"强于熊市的"下压力"（长期沪深300 偏多的体现）。
- **波动率**：熊市标准差 1.65% > 牛市 1.42%，且熊市 P1 -5.17% 比牛市 P1 -4.19% 更深；
  **熊市更躁动**，暴跌日更频繁（fat left tail）。
- **右尾**：P99 牛市 +4.06% ≈ 熊市 +4.29%，几乎相当——
  熊市里的"反弹日"涨幅不亚于牛市的"拉升日"（V 型反转常见）。
- **极端段**：
  - **#3（2008 金融危机熊）** 标准差 2.89%、P1 -7.2%、P99 +8.0%，全程极度躁动。
  - **#7（2015 股灾熊）** P1 -8.05%，样本里最极端的单日跌。
  - **#2（2005-2007 史诗牛）** 日均 +0.36%（年化 ~+140%），波动也大。
  - **#8（2016-2018 慢牛）** 标准差仅 0.79%，是样本里最温和的牛市——
    整段只涨 52%，但走势平稳。
