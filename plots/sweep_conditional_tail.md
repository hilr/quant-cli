# sweep_conditional_tail — 跨窗口扫描条件分布，筛选厚尾可操作模式

`sweep_conditional_tail.py` 的文档与结果。**批量扫描** `N1=1..10 × N2=1..3 × 牛/熊 × 10 个收益桶`
共 ~600 个条件分布，按"厚尾 + 显著均值偏移"过滤，挑出真正可操作的模式——
而不是只看单一组合。

和 `conditional_forward_return` 的关系：后者看单个 (N1, N2) 组合，
本工具**跨组合扫描**，找出全局最优的条件模式。

## 用法

```bash
# 默认扫描 N1=1..10 × N2=1..3，输出 top 12 box plot
uv run python plots/sweep_conditional_tail.py

# 调大 top N（例如双向各 8 个 = 共 16 个）
uv run python plots/sweep_conditional_tail.py --top 16

# 自定义输出路径
uv run python plots/sweep_conditional_tail.py --output /mnt/dataset/my_sweep.png
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--top` | box plot 显示的模式数（正负各一半） | `12` |
| `--output` | 输出 PNG 路径 | `/mnt/dataset/sweep_conditional_tail.png` |

> 桶边界固定为 `conditional_forward_return` 的默认值
> `[-∞, -8%, -5%, -3%, -1%, 0%, +1%, +3%, +5%, +8%, +∞]`，
> 想改桶需修改源码 `DEFAULT_BUCKET_EDGES`（或先扩展 CLI）。

**输出：**
- 控制台：总览计数 + 按 |均值偏移| 降序的 top 25 + 按 kurt 降序的 top 25
- PNG：双向 box plot，左 = 显著负偏移、右 = 显著正偏移，分布信息卡片在 x 轴下方

## 筛选标准

进入候选池需要同时满足：

| 条件 | 阈值 | 含义 |
|------|------|------|
| 样本量 | `n >= 30` | 统计可靠性下限 |
| 厚尾 | `excess kurtosis >= 1.0` | 正态=0；>1 视为明显厚尾 |
| 显著偏移 | `桶均值 95%CI 不含基线均值` | `1.96 × std/√n` 估计 |

546 个 n≥30 桶中，440 个厚尾、90 个厚尾+显著偏移。box plot 取正负偏移
按 |shift| 各 top N 显示。

## 图说明

- **左右排列**：左半 = 显著负偏移（最负在最左）、右半 = 显著正偏移（最正在最右）。
- **箱线图**：须=P5/P95、箱=P25/P75、黑横线=中位。
- **彩点**：桶均值（牛红、熊绿）。
- **灰短横 `_`**：该 regime/N1/N2 的全局基线均值（参考线，灰短横与彩点的距离=均值偏移）。
- **x 轴下方卡片**：每个模式的分布信息——
  regime / N1→N2 / 桶边界 / n / μ / Δ / P5 / P95 / kurt / skew。
- **边框颜色**：按 regime 染色（牛红、熊绿），快速识别 regime 分布。

## 怎么读

| 看点 | 含义 |
|------|------|
| 红点（均值）远高于灰短横（基线） | 该条件下未来收益显著上行 → 买入信号 |
| 红点远低于灰短横 | 该条件下未来收益显著下行 → 规避/做空信号 |
| 正负两侧 shift 幅度对比 | 机会侧 vs 风险侧的不对称性 |
| 卡片 kurt > 3 | 该条件下分布厚尾明显，极端事件概率高于正态 |
| 卡片 |skew| > 0.5 | 分布形状不对称，偏向某一尾 |
| 牛市桶 vs 熊市桶数量 | 哪个 regime 的可操作信号更多 |
| 同一桶跨 N1 的稳定性 | 信号是否随窗口长度变化（越稳越可信） |

## 解读

### 机会侧：牛市右尾占主导

- **Pattern 1（牛 1d→3d 暴涨）**：全表最强信号，单日 +3-5% 暴涨后未来 3 日 +2.05%、
  P95 +7.22%、kurt 5.11。**牛市单日暴涨是最值得策略化的延续信号**。
- **Pattern 2（牛中期上涨）**：N1=3-10、桶 [+5%, +8%) 都显示稳定正偏移，
  跨 N1 稳健，n 大、P95 普遍 6-8%。
- **Pattern 3（牛长期超跌反弹）**：N1=8-10、桶 (-∞, -8%] 显示反弹 +1.4-1.6%。
- **唯一的熊市代表**：N1=5、桶 [+8%, +∞) ——熊市里短期暴涨后仍有 +0.78%
  （基线 -0.31%），但 CI 边缘显著、n=39 偏小。

### 风险侧：牛市横盘陷阱

- **所有 8 个显著负偏移都在牛市 + 低波动桶**（前 N1 日收益 ∈ [-1%, +1%)）。
  牛市里横盘 ≠ 蓄势，而是后续跑输基线 0.18-0.39pp 的信号。
- **互为印证**：牛市真正赚钱的是有大方向的日子（Pattern 1/2/3），
  没方向就空仓比硬上车好。

### 整体不对称

- **机会侧 shift 幅度（+0.4 ~ +1.5pp）整体大于风险侧（-0.18 ~ -0.39pp）**——
  机会比风险更剧烈。
- **风险侧几乎全是高 kurt（5-8）**——厚尾风险，规避价值更高。
- **熊市两端都不强**——熊市择时信号本来就稀少，整体基线下压占主导。

### 实盘含义

| 排名 | 信号 | 触发 | 持仓 | 期望 | 风险（P5） |
|---|---|---|---|---|---|
| ⭐1 | 牛市单日暴涨延续 | regime=牛 & r_back(1d) ∈ [+3%, +5%) | 3 日 | +2.05% | -2.29% |
| ⭐2 | 牛市中期上涨延续 | regime=牛 & r_back(6d) ∈ [+5%, +8%) | 3 日 | +1.10% | -3.65% |
| 3 | 牛市长期超跌反弹 | regime=牛 & r_back(10d) ≤ -8% | 2 日 | +1.63% | -3.05% |
| 4 | 牛市横盘规避 | regime=牛 & r_back(3d) ∈ [+0%, +1%) | 3 日 | +0.18% | -3.16% |

## 局限

- **聚合统计的局限**：top 模式按 |shift| 选，会偏向**样本中极端值**。
  对小样本桶（n=30-70）的均值估计功效有限，95%CI 较宽——
  看偏移幅度时同时看 CI 范围。
- **桶边界固定**：默认 10 桶覆盖 ±8% 之外，对极端尾部（如 ±15% 以上）样本稀少。
  关注更极端的尾部需修改 `DEFAULT_BUCKET_EDGES`。
- **同 `conditional_forward_return` 的局限**：重叠窗口自相关、跨段边界窗口不剔除、
  最后一段未完结。多重比较（90 个候选桶）未做 Bonferroni 等校正，
  实际显著性会被高估——参考时按 effect size + 复现稳定性判断更可靠。
