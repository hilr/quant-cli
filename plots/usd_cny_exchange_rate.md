# 人民币兑美元汇率（USD/CNY 月度中间价）vs 沪深300

对应脚本：`plots/usd_cny_exchange_rate.py`
输出：`/mnt/dataset/usd_cny_exchange_rate.png`

## 图说明

- **上图**：USD/CNY 汇率（人民币元/美元）
  - 左轴：月末中间价（eop，蓝实线）+ 月均（avg，蓝虚线），8.28 盯住线作虚线基准。
  - 右轴：沪深300收盘（红，淡）。
- **下图**：月末中间价环比变动（%），红=贬值（数值变大），绿=升值。
- 时间范围 1999 至今。
- 重大汇率制度节点用箭头标注（见下）。

## 算法

```
mom[t] = usd_cny_eop[t] / usd_cny_eop[t−1] − 1
```

直接读 `pbc/exchange_rate.csv` 的 `usd_cny_eop`（月末）/ `usd_cny_avg`（月均）两列，与沪深300日收盘按日期 join（沪深300仅 2005 后，故早期汇率段无指数对照）。

## 怎么读

| 信号 | 含义 |
|---|---|
| **USD/CNY 走高（数值变大）** | 人民币贬值，资本外流压力，通常压制 A 股 |
| **USD/CNY 走低（数值变小）** | 人民币升值，外资流入预期，利好 A 股 |

## 重大汇率制度节点（脚本标注）

| 时点 | 事件 |
|---|---|
| 2005-07 | 7/21 汇改，脱钩美元，开启渐进升值 |
| 2008-07 | 金融危机重新盯住 ~6.83 |
| 2010-06 | 重启浮动 |
| 2014-01 | 升值高点 ~6.04 |
| 2015-08 | 811 汇改，一次性贬值 |
| 2022-04 | 快速贬值 6.3→7.3 |

## 与沪深300的关系

- 人民币升值周期（2005-2014）伴随外资流入与 A 股牛市；
- 急速贬值（2015-08 811 汇改、2022-04）往往对应 A 股调整与外资流出；
- 汇率是外资进出 A 股的重要传导变量。

## 数据源

- `pbc/exchange_rate.csv`：`usd_cny_eop`（月末中间价）/ `usd_cny_avg`（月均中间价）。
- `index_quote_history/000300.parquet`：日收盘。
