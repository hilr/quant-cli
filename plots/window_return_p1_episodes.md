# window_return_p1_episodes — N 日窗口 P 分位极端事件的时间分布

对应脚本：`plots/window_return_p1_episodes.py`
无图输出，只在终端打印按月聚合的事件分布表。

## 图说明

对每个标的、每个窗口长度，挑出历史上所有「≤ Pq 分位阈值」的窗口，
按"窗口结束日"所在月份聚合，回答：

> 「历史上极端的 N 日跌幅（或涨幅）都发生在哪些月份？是不是高度
>   集中在某几次危机？」

每个标的输出三件事：
- **阈值**：`P{percentile}` 分位对应的收益水平（左尾跌 / 右尾涨）。
- **极端窗口数 / 总样本数**：触发该分位的窗口数。
- **按月聚集**：每个 YYYY-MM 触发该分位的窗口数（按时间序）。
- **当前窗口**：最新一个 N 日窗口的收益和经验百分位 p。

## 算法

- **收益率**：`close[t]/close[t-N] - 1`，与 `window_return_distribution` 同口径。
  指数用原始 close、基金/股票建议传前复权目录。
- **阈值**：`numpy.percentile(r, q)`（默认线性插值）。
- **筛选**：左尾 `ret ≤ threshold`，右尾 `ret ≥ threshold`。
  严格不等号让约 N×q% 个窗口被圈进来（具体个数受插值与重复值影响）。
- **滚动重叠窗口（step=1）**：每个交易日都开一个新窗口。
  ⚠️ 相邻窗口高度自相关，做"事件分布"参考可以，
  做严格统计推断（如置信区间）要谨慎。
- **当前窗口**：取最新一个交易日的窗口收益。

## 怎么读

| 看点 | 含义 |
|------|------|
| 极端事件**高度集中在某几个月** | 说明该标的的极端跌幅由"少数危机事件"驱动，可用作"危机清单" |
| 极端事件**散布在很多月份** | 说明该标的常态高波动，"极端"并不那么稀罕 |
| 当前 p 在阈值附近 | 当前 N 日收益处于历史极端位置，需要警惕 / 关注反弹机会 |
| 当前窗口的月份**新进入清单** | 本轮行情已经在历史上留下印记 |

## 用法

```bash
# 默认：5 大 A 股宽基/风格指数 × 5/10/20 日窗口 × P1（左尾）
uv run python plots/window_return_p1_episodes.py

# 只看某个窗口
uv run python plots/window_return_p1_episodes.py --window 10

# 看多个窗口
uv run python plots/window_return_p1_episodes.py --window 5 10 20 60

# 换分位（P5）
uv run python plots/window_return_p1_episodes.py --percentile 5

# 右尾（极端涨）
uv run python plots/window_return_p1_episodes.py --direction upper

# 自定义标的（默认 5 个指数）
uv run python plots/window_return_p1_episodes.py \
  --codes 000300:沪深300 000905:中证500

# 换行情目录（基金/股票前复权）
uv run python plots/window_return_p1_episodes.py \
  --adjusted-dir /mnt/dataset/fund_quote_adjusted \
  --codes 512890:中证A500ETF
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--adjusted-dir` | 含 `{code}.parquet` 的行情目录 | /mnt/dataset/index_quote_history |
| `--window` | 窗口长度（交易日），可传多个 | 5 10 20 |
| `--percentile` | 分位阈值（0-100） | 1 |
| `--direction` | lower=左尾 / upper=右尾 | lower |
| `--codes` | 自定义标的，格式 `code:name` | 5 大 A 股宽基/风格指数 |

## 数据源

- `{adjusted-dir}/{code}.parquet`：标的的日收盘（`date`, `close` 两列即可）。

**几个判读要点**：

1. **沪深300/中证500/中证1000 的 10d P1 高度集中在 2008 + 2015~2016**，两轮
   就占总数的 80% 以上——这类宽基的极端左尾几乎完全由"系统性危机"定义。
2. **创业板指的 10d P1 几乎只在 2015 股灾 + 2016 熔断**（31/40），开板于 2010、
   错过了 2008，"历史极值"参考系较窄。
3. **科创50 的 10d P1 由 5 段独立事件构成**（疫情 / 牛市急涨 / 美债 / 封城 /
   量化踩踏 / 关税 / 2026-06~07），开市仅 5 年，统计意义弱于其他指数。
4. **本轮 2026-07 已被记入历史 10d P1**：创业板指、科创50 当月各新增 1 个
   极端窗口——本月行情已经在历史上留下印记。
