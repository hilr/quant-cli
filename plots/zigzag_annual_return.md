# zigzag_annual_return 分析结果

`zigzag_annual_return.py` 的分析结果快照。本文件随数据更新，由脚本控制台输出誊写。

## 方法

1. 用 `pivot_eval.zigzag_pivots`（15% 反转阈值）在沪深300 day high/low 上标记阶段高/低点（H/L 枢轴）。
2. 提取所有 L→H 上升段（做多机会）。做多 = 在 L 枢轴日买入、H 枢轴日卖出。
3. **日历年边界切分**：每个上升段在跨年处切开。年尾最后一个交易日强制卖出、年头第一个交易日重新买入，相邻年份不共享边界，跨年跳空不计入任何一年。
4. **未完结持仓**：最后一个枢轴若为 L（至今未反转），按 L→最新收盘价作为「持仓中」波段纳入——历史年份段用实际收盘价（已确认），当前年份段用最新收盘价（mark-to-market）。
5. 收益用实际收盘价计算（枢轴极值 high/low 仅用于定位买卖点）。每年取该年所有段中的最大收益。
