# zigzag_annual_return — ZigZag 枢轴每年潜在最大收益与潜在最大损失（沪深300）

`zigzag_annual_return.py` 的完整文档与分析结果快照。结果部分随数据更新，由脚本控制台输出誊写。

## 用法

```bash
# 沪深300，15% 阈值
uv run python plots/zigzag_annual_return.py

# 只打印表格不画图
uv run python plots/zigzag_annual_return.py --pct 0.15

# 改阈值 + 输出图
uv run python plots/zigzag_annual_return.py --pct 0.10 --output /mnt/dataset/zigzag_annual_return_000300.png
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--index-dir` | 指数 parquet 父目录 | /mnt/dataset/index_quote_history |
| `--code` | 指数代码 | 000300（沪深300） |
| `--pct` | ZigZag 反转阈值 | 0.15 |
| `--output` | 输出 PNG 路径（不指定则只打印表格） | None |

**输出：** 控制台年度表格（潜在最大收益 + 潜在最大损失各一份） + 三面板 PNG（上=价格+枢轴+上升段高亮，虚线为持仓中波段；中=每年潜在最大收益柱形；下=每年潜在最大损失柱形）。

## 方法

每年柱子只取**一段**——那年能找到的最佳/最差单个 zigzag 波段。同年多段不合并、不复利、不求和，段间空仓收益为 0 不参与计算。

1. 用 `pivot_eval.zigzag_pivots`（15% 反转阈值）在沪深300 day high/low 上标记阶段高/低点（H/L 枢轴）。
2. **潜在最大收益**：提取所有 L→H 上升段（最低点买入、相邻最高点卖出）。每年取该年所有上升段中涨幅最大的一段。
3. **潜在最大损失**：提取所有 H→L 下降段（最高点买入、相邻最低点卖出）。每年取该年所有下降段中跌幅最深的一段。
4. **日历年边界切分**：每个段在跨年处切开。年尾最后一个交易日强制卖出/平仓、年头第一个交易日重新买入/重启基点，相邻年份不共享边界，跨年跳空不计入任何一年。
5. **未完结持仓**：
   - 上升段：最后一个枢轴若为 L（至今未反转），按 L→最新收盘价作为「持仓中」波段纳入。
   - 下降段：最后一个枢轴若为 H（至今未反转），按 H→最新收盘价作为「持仓中」波段纳入。
6. 收益用实际收盘价计算（枢轴极值 high/low 仅用于定位买卖点）。

## 未完结持仓校验（#30 ⚡）

- 买入 2025-04-07（close 3589.44），至今未反转，总收益 = 4943.02 / 3589.44 − 1 = **37.71%**
