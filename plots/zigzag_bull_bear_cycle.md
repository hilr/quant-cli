# zigzag_bull_bear_cycle — ZigZag 枢轴牛熊周期（沪深300）

`zigzag_bull_bear_cycle.py` 的完整文档与结果快照。结果随数据更新，由脚本控制台输出誊写。

## 用法

```bash
# 沪深300，30% 阈值（默认）
uv run python plots/zigzag_bull_bear_cycle.py

# 输出图
uv run python plots/zigzag_bull_bear_cycle.py --output /mnt/dataset/zigzag_bull_bear_cycle_000300.png

# 改阈值/指数
uv run python plots/zigzag_bull_bear_cycle.py --pct 0.20 --code 000905
```

| 参数 | 说明 | 默认 |
|------|------|------|
| `--index-dir` | 指数 parquet 父目录 | /mnt/dataset/index_quote_history |
| `--code` | 指数代码 | 000300（沪深300） |
| `--pct` | ZigZag 反转阈值 | 0.30 |
| `--output` | 输出 PNG 路径（不指定则只打印表格） | None |

**输出：** 控制台周期明细表 + 牛熊汇总 + 双面板 PNG（上=价格+枢轴+牛绿熊红区间阴影；下=各周期历时横条）。

## 方法

1. `pivot_eval.zigzag_pivots`（30% 反转阈值）在沪深300 day high/low 上标记阶段高/低点（H/L 枢轴，回顾式，最后一个未确认极值不计）。
2. 相邻枢轴配成一段：**L→H = 牛市（上涨段）**，**H→L = 熊市（下跌段）**。
3. 每段记录起止日期、价格、涨跌幅、日历天数、年数、交易日数。
4. 最后一段（最后枢轴→最后一日）标 ⚡ 未完结，不纳入汇总统计（实时不可知是否反转）。

## 解读

- 30% 阈值滤掉中小波动，只保留大级别周期——16 段（7 牛 8 熊），每段都是真实的大趋势。
- **牛熊时长接近**：牛市均 551 天 vs 熊市均 526 天，几乎对称；但**牛市涨幅远超熊市跌幅**（+164% vs -44%），长周期沪深300 强多头。
- 三轮大牛市：2005-2007（**+629%**）、2012-2015（+156%）、2019-2021（+102%）。
- 两轮深熊：2008 金融危机（-60.58%）、2021-2024（-47.59%，持续近 3 年）。
- **当前**（#16⚡）：自 2024-02-02 低点 3108 起涨至 4919，**+58.26%**、历时 872 天仍未触发 30% 反转 → 处于未完结大牛市中。
